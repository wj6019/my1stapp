# 天气查询工具

这是一个基于 Flask 和和风天气 API 的天气查询工具，包含后端逻辑和前端界面。

## 功能特点
- 支持通过城市名称查询未来三天的天气预报。
- 提供美观的前端界面，支持动态查询。
- 使用和风天气 API 获取实时天气数据。

## 技术栈
- 后端：Python、Flask
- 前端：HTML、CSS、Bootstrap、jQuery
- 外部服务：和风天气 API
